# Harmony Care Knowledge Base

## Overview

This knowledge base contains comprehensive regulatory information for assisted living facilities across the United States. Each state overview provides detailed information about licensing requirements, administrator qualifications, staffing requirements, training mandates, and regulatory compliance.

## Structure

```
knowledge-base/
├── USA/
│   ├── [State]_Overview.md
│   └── ...
└── README.md
```

## State Overviews

Each state overview document includes:

- **Executive Summary** - Key regulatory highlights and unique features
- **Regulatory Authority and Structure** - Primary agencies and legal framework
- **Facility Licensure** - License types, terms, and requirements
- **Facility Scope of Care** - Services provided and limitations
- **Resident Assessment and Service Planning** - Assessment requirements and processes
- **Medication Management** - Medication administration policies and procedures
- **Administrator Requirements** - Qualifications, training, and licensing
- **Staffing Requirements** - Ratios, qualifications, and scheduling
- **Direct Care Staff Training** - Initial and continuing education requirements
- **Quality Requirements** - Quality assurance and improvement programs
- **Infection Control** - Policies and procedures
- **Emergency Preparedness** - Disaster planning and response
- **Medicaid and Financial Assistance** - Waiver programs and funding sources
- **Key Distinguishing Features** - 10 unique aspects of each state's regulations
- **Resources and Citations** - Primary sources and regulatory references
